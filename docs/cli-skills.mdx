## Core commands

### `skills validate`

```bash
agentpack skills validate [target]
```

Checks:

- compiler-mode `SKILL.md` structure
- source bindings exist
- skill imports align with package dependencies
- package metadata is publishable

### `skills inspect`

```bash
agentpack skills inspect <target>
```

Shows the resolved package, source bindings, imported skills, and compiled metadata.

### `skills build`

```bash
agentpack skills build <target>
```

Produces `.agentpack/compiled.json`.

### `skills materialize`

```bash
agentpack skills materialize
```

Reads `.agentpack/compiled.json` and emits runtime outputs plus `.agentpack/materialization-state.json`.

### `skills dev`

```bash
agentpack skills dev <target>
```

Compiles, materializes, watches for changes, and starts the localhost workbench. Use `--no-dashboard` to skip the browser UI.

### `skills stale`

```bash
agentpack skills stale [target]
```

Reads compiled state and reports changed sources, stale skills, and affected downstream skills.

### `skills install`

```bash
agentpack skills install <package>
```

Resolves packages, updates install state, compiles the resulting graph, and materializes runtime outputs.

### `skills status`

```bash
agentpack skills status
```

Shows environment health, runtime drift, deprecated packages, and registry configuration.
