---
title: "Generated Files Reference"
description: "Reference for the generated state files agentpack writes under .agentpack/."
---

agentpack manages several state files under the `.agentpack/` directory. Some are meant to be committed, others are runtime-only.

## Summary

| File | Produced by | Commit? | Purpose |
|---|---|---|---|
| `compiled.json` | `skills build`, `skills validate` | **Yes** | Semantic source of truth: compiled skills, source hashes, graph edges |
| `install.json` | `skills install`, `skills uninstall` | No | Tracks direct and transitive package installs |
| `materialization-state.json` | `skills materialize`, `skills install`, `skills dev` | No | Tracks emitted runtime files and adapter ownership |
| `dev-session.json` | `skills dev` | No | Active dev session PID and linked skill paths |

---

## `.agentpack/compiled.json`

The semantic source of truth for the skill graph. Contains compiled skills, source content hashes, body reference occurrences, and aggregated graph edges.

**Produced by:** `agentpack skills build <target>`, `agentpack skills validate <target>`

**Structure:**

```json
{
  "rootSkill": "brand-copywriting",
  "skills": {
    "brand-copywriting": {
      "name": "brand-copywriting",
      "description": "Write brand-consistent copy...",
      "skillImports": { ... },
      "sourceBindings": { ... },
      "occurrences": [ ... ],
      "edges": [
        {
          "source": "skill:brand-copywriting",
          "target": "domains/brand/knowledge/tone-of-voice.md",
          "kind": "source_usage",
          "labels": ["tone guidance"],
          "contexts": ["primary source of brand voice constraints"],
          "occurrenceCount": 1
        }
      ]
    }
  },
  "sourceHashes": {
    "domains/brand/knowledge/tone-of-voice.md": "sha256:a1b2c3..."
  }
}
```

**Commit this file.** It provides the baseline hashes for [staleness detection](/staleness) and the graph data for the dashboard. Without it, `skills stale` has nothing to compare against.

## `.agentpack/install.json`

Consumer-side install state. Tracks which skill packages are installed, whether they are direct or transitive, and their resolved versions.

**Produced by:** `agentpack skills install`, `agentpack skills uninstall`

**Structure:**

```json
{
  "installs": {
    "@acme/brand-copywriting": {
      "direct": true,
      "package_version": "1.2.0",
      "skills": [
        { "name": "brand-copywriting", "skillFile": "SKILL.md" }
      ]
    },
    "@acme/editorial-principles": {
      "direct": false,
      "package_version": "2.0.1",
      "skills": [
        { "name": "editorial-principles", "skillFile": "SKILL.md" }
      ]
    }
  }
}
```

**Do not commit.** This is runtime state specific to the consumer repo. Each repo has its own install set.

## `.agentpack/materialization-state.json`

Tracks which runtime files agentpack emitted and which adapter owns each one. Used for cleanup when skills are uninstalled or when the materialization target changes.

**Produced by:** `agentpack skills materialize`, `agentpack skills install`, `agentpack skills dev`

**Structure:**

```json
{
  "rootSkill": "brand-copywriting",
  "outputs": {
    "claude": [
      { "target": ".claude/skills/brand-copywriting", "mode": "symlink" }
    ],
    "agents": [
      { "target": ".agents/skills/brand-copywriting", "mode": "symlink" }
    ]
  }
}
```

**Do not commit.** Runtime adapter state that is repo-specific and regenerated by `skills materialize`.

---

## `.agentpack/dev-session.json`

Tracks the active `skills dev` session: the process PID, the root skill, and all linked skill paths. Used for crash recovery and cleanup.

**Produced by:** `agentpack skills dev`

**Structure:**

```json
{
  "pid": 12345,
  "rootSkill": "brand-copywriting",
  "links": [
    ".claude/skills/brand-copywriting",
    ".agents/skills/brand-copywriting"
  ]
}
```

**Do not commit.** Ephemeral session state. If a dev session crashes, run `agentpack skills dev cleanup` to remove stale links.

---

## `.gitignore` recommendation

```gitignore
# agentpack — commit compiled.json, ignore the rest
.agentpack/install.json
.agentpack/materialization-state.json
.agentpack/dev-session.json
```
