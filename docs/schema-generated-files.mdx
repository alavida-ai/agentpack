---
title: "Generated Files Reference"
description: "Reference for the generated state files agentpack writes under .agentpack/."
---

## `.agentpack/compiled.json`

- semantic source of truth
- produced by `skills build` and compiler-mode `skills validate`
- contains compiled skills, source hashes, occurrences, and graph edges

## `.agentpack/install-state.json`

- consumer/runtime install state
- tracks direct and transitive package installs
- do not commit

## `.agentpack/materialization-state.json`

- runtime adapter ownership
- tracks emitted `.claude/skills/` and `.agents/skills/` outputs
- do not commit

## `.agentpack/dev-session.json`

- active `skills dev` session ownership
- used for cleanup and crash recovery
- do not commit
